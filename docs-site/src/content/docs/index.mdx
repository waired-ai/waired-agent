Waired is a private, inference-only overlay network. It connects your laptops,
GPU servers, VMs, and home boxes so that **local LLMs can be shared across your
own devices** without exposing them to the public internet. Type in Claude Code
on any laptop and the answer comes from your own machine — no per-message bill,
no subscription, and your prompts never leave your devices.

<CardGrid>
	<Card title="One command" icon="rocket">
		Install with a single line, sign in with Google, and your coding agent is
		linked automatically. See [Install](/getting-started/install/).
	</Card>
	<Card title="Free to run" icon="approve-check">
		The model runs on hardware you already own — a recent CPU handles a 7B
		model, and a GPU makes it fast.
	</Card>
	<Card title="Private by design" icon="seti:lock">
		Peers talk directly over an end-to-end encrypted WireGuard link. The
		coordination service never sees your prompts or replies — even the relay
		can't read them. See [Privacy](/concepts/privacy/).
	</Card>
	<Card title="Your tools, unchanged" icon="seti:default">
		Works with Claude Code and OpenCode out of the box, plus any
		OpenAI- or Anthropic-compatible client.
	</Card>
</CardGrid>
