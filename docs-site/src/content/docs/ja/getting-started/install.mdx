Waired は 3 つの OS すべてで、**コマンド 1 つ**で同じようにインストールできます。
ターミナルに 1 行貼り付けると、何をするかが表示され、確認してから実行されます。
ダウンロードしてダブルクリックする形式のインストーラーはありません。

<Aside type="tip" title="ターミナルを使ったことがなくても、貼り付けるだけです">
	各 OS のページは「ターミナルの開き方」から始まり、コマンドには必ず
	「何をするコマンドか」の 1 文が添えてあります。この 1 コマンドさえ済めば、
	あとは画面（ブラウザとアイコン）だけでも進められます —
	[クイックスタート](/ja/quickstart/)を参照してください。
</Aside>

## 自分のパソコンで動きますか？

| 役割 | 対応 |
|---|---|
| **AI を動かす** | Ubuntu 24.04 以降 / Debian trixie 以降 / Windows 11・Server 2025 / macOS 13 以降（Apple Silicon 推奨） |
| **AI を使う** | 上記のいずれか — グラフィックボードは不要 |

グラフィックボードがあると回答は大幅に速くなりますが、最近の CPU でも小さいモデルなら
問題なく動きます。入力に使うだけのパソコンには特別な性能は要りません。

どのモデルにどれだけメモリが必要かは[モデルカタログ](/ja/reference/model-catalog/)に
ありますが、セットアップ時に収まるモデルを Waired が選ぶので、読まなくても大丈夫です。

## インストーラーが置くもの

- **Waired 本体** — このパソコンを接続し続ける常駐サービスと、入力できる `waired`
  コマンド。サービスは、誰もログインしていなくても電源投入時から動きます。
- **アイコン** — メニューバー（macOS）またはタスクトレイ（Windows / Linux）。
  日常の操作はここから行えます。
- **それ以外はまだ何も。** AI ソフト本体（数 GB）はセットアップ時に、
  「このパソコンでモデルを動かす」と答えた場合にだけ導入されます。

どのインストーラーも、実行前に内容を表示して確認を求めます。

<Aside type="note" title="すでに Ollama を使っている場合">
	既存のインストールに触れないようにできます。
	[高度なインストールオプション](/ja/reference/install-options/)を参照してください。
</Aside>
