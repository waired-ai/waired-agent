どこかのノートパソコンで Claude Code に入力すると、その答えは**自分が持っているマシン**から
返ってきます。メッセージごとの課金もサブスクリプションもなく、プロンプトが自分のデバイスから
外に出ることもありません。

## 選ばれている理由

<CardGrid>
	<IconLinkCard title="コマンド 1 つ" icon="rocket" href="/ja/quickstart/">
		1 行でインストールし、Google でサインイン。あとは画面でもターミナルでも
		仕上げられます — どちらでも 10 分ほどです。
	</IconLinkCard>
	<IconLinkCard title="動かすのは無料" icon="approve-check" href="/ja/reference/model-catalog/">
		モデルはすでに持っているハードウェアで動きます。最近の CPU でも動き、
		グラフィックボードがあれば高速です。自分のマシンに何が載るか見てください。
	</IconLinkCard>
	<IconLinkCard title="設計からプライベート" icon="padlock" href="/ja/concepts/privacy/">
		デバイス同士がエンドツーエンド暗号化された経路で直接通信します。
		Waired 側のサーバーがプロンプトや応答を見ることはありません。
	</IconLinkCard>
	<IconLinkCard title="ツールはそのまま" icon="puzzle" href="/ja/guides/coding-agents/">
		Claude Code、OpenCode、OpenAI 互換クライアントはそのまま動きます。
		変わるのは答えの出どころであって、作業のしかたではありません。
	</IconLinkCard>
</CardGrid>

## コマンド 1 つでインストール

<Tabs syncKey="os">
	<TabItem label="Windows">
		**ターミナル（管理者）**から — スタートボタンを右クリック:

		```powershell
		iwr -useb https://github.com/waired-ai/waired-agent/releases/latest/download/install.ps1 | iex
		```
	</TabItem>
	<TabItem label="macOS">
		**ターミナル**から — ⌘ + スペースで「ターミナル」と入力:

		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```
	</TabItem>
	<TabItem label="Linux">
		ターミナルから（Ubuntu 24.04 以降 / Debian trixie 以降）:

		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```
	</TabItem>
</Tabs>

何を行うかを表示し、変更前に必ず確認します。そのあとのセットアップは
**画面でもターミナルでも**進められます — [クイックスタート](/ja/quickstart/)が両方を扱います。
