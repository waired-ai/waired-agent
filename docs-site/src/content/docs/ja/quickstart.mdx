このページを最後まで進めると、**自分のパソコンで動く AI** ができあがり、
Claude Code がクラウドではなくその AI に質問するようになります。

## はじめる前に

|  | |
|---|---|
| **かかる時間** | 操作は 10 分ほど。そのあとダウンロードの待ち時間が 5〜30 分あります（放置で OK）。 |
| **必要なもの** | AI を動かすパソコン 1 台（Windows 11 / macOS 13 以降 / Ubuntu 24.04 以降・Debian trixie 以降）、Google アカウント、そのパソコンの管理者権限。 |
| **なくてもいいもの** | グラフィックボード（GPU）。あれば速くなりますが、最近の CPU でも小さめのモデルなら動きます。 |

<Aside type="tip" title="どのパソコンで動かせばいい？">
	いちばん性能の高い 1 台を選んでください（グラフィックボード搭載のデスクトップ、
	または Apple Silicon の Mac）。ノートパソコンは、あとから**どこにいても**
	そのパソコンの AI を使えるようになります。
	パソコンが 1 台しかない場合は、その 1 台で大丈夫です。
</Aside>

<Steps>

1. ### Waired をインストールする

	<Tabs syncKey="os">
		<TabItem label="Windows">
			**インストーラーをダウンロード**して、ダブルクリックします。

			<LinkCard
				title="WairedSetup-x64.exe"
				href="https://github.com/waired-ai/waired-agent/releases/latest"
				description="最新リリースページの WairedSetup-<バージョン>-x64.exe というファイルです。"
			/>

			「このアプリがデバイスに変更を加えることを許可しますか？」という
			青い画面（ユーザーアカウント制御）が出たら、**「はい」**を選びます。

			<Aside type="caution" title="「WindowsによってPCが保護されました」と出たら">
				現在インストーラーにコード署名がないため、初回だけこの警告が出ます。
				**「詳細情報」**をクリックし、**「実行」**を押してください。
			</Aside>

			<details>
			<summary>コマンドで入れたい場合（PowerShell）</summary>

			```powershell
			iwr -useb https://github.com/waired-ai/waired-agent/releases/latest/download/install.ps1 | iex
			```

			</details>
		</TabItem>
		<TabItem label="macOS">
			macOS 用のダブルクリック式インストーラーはまだありません。
			**ターミナル**を開き（⌘ + スペース →「ターミナル」と入力 → Enter）、
			次の 1 行を貼り付けて Enter を押します。

			```sh
			curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
			```

			途中で Mac のパスワードを聞かれます。入力しても画面には何も表示されませんが、
			故障ではありません。そのまま入力して Enter を押してください。
		</TabItem>
		<TabItem label="Linux">
			ターミナルを開き、次の 1 行を貼り付けて Enter を押します。

			```sh
			curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
			```

			途中でパスワードを聞かれます。入力しても画面には何も表示されませんが、
			故障ではありません。

			<Aside type="note">
				対応しているのは Ubuntu 24.04 以降と Debian trixie 以降です。
				→ [Linux へのインストール](/ja/getting-started/install/linux/)
			</Aside>
		</TabItem>
	</Tabs>

	インストーラーは何も変更する前に「これから何をするか」を表示し、
	`Proceed? [Y/n]` と確認します。Enter を押せば続行します。

2. ### セットアップを始める

	自分で入力するコマンドはこれだけです。この先はブラウザで進みます。

	<Tabs syncKey="os">
		<TabItem label="Windows">
			**管理者として PowerShell を開き**（スタートボタンを右クリック →
			「ターミナル（管理者）」）、次を実行します。

			```powershell
			& "C:\Program Files\Waired\waired.exe" init
			```
		</TabItem>
		<TabItem label="macOS">
			```sh
			sudo waired init
			```
		</TabItem>
		<TabItem label="Linux">
			```sh
			sudo waired init
			```
		</TabItem>
	</Tabs>

	サインイン用のリンクが表示され、そこで止まります。**Enter** を押すと
	ブラウザーが開くので、Google アカウントを選んでください。

	<Aside type="caution" title="このウィンドウは閉じないでください">
		セットアップはブラウザに移りますが、管理者権限が必要な作業（AI ソフトの導入、
		コーディングツールの連携）はこのウィンドウが担当し続けます。

		誤って閉じてもデータは失われません。ブラウザに
		*「The setup command on … was closed before this finished. Your progress was saved.」*
		と表示されるので、同じコマンドをもう一度実行すれば続きから再開できます。
	</Aside>

3. ### ブラウザでセットアップを終える

	サインインすると、いま設定中のパソコンのセットアップ画面に移動します。
	3 つのステップが自動で進みます。

	<Screenshot
		src="setup-wizard-progress.png"
		alt="ブラウザに表示された Waired のセットアップ画面。3 つのステップ（Install the AI software = 完了、Download the AI model = 進行中で 19.2 GB 中 7.9 GB、Check the speed = 待機中）が並んでいる。"
		caption="パソコン自身が報告している進捗です。このタブを閉じてもダウンロードは止まりません。"
	/>

	| ステップ | 内容 | 目安 |
	|---|---|---|
	| **Install the AI software** | モデルを動かすプログラムの導入。管理者権限が必要なため、ターミナルを開いたままにする必要があります。 | 1〜2 分 |
	| **Download the AI model** | AI 本体。このパソコンで実際に動かせる最良のモデルを Waired が選びます。 | 5〜30 分 |
	| **Check the speed** | 速度の簡易計測。 | 最大 5 分 |

	最後にコーディングツールの連携を聞かれるので、**Claude Code** を有効にします。

	<Aside type="note" title="セットアップ画面は英語表示です">
		Web 画面は現在英語のみです。日本語化は予定されています。
	</Aside>

4. ### 試しに話しかける

	ターミナルに戻って:

	```sh
	waired infer "say hi"
	```

	数秒であいさつが返ってきます。この返事は**あなたのパソコンの中だけ**で
	作られたもので、クラウドの AI サービスには何も送られていません。

	<Aside type="note" title="何も返ってこないとき">
		`waired doctor` を実行してください。設定を上から順に点検して ✓ / ⚠ / ✗ で表示し、
		**f** キーを押すと直せるものは自動で直します。
	</Aside>

5. ### Claude Code から使う

	いつもどおり Claude Code を開いて質問してみてください。
	画面下部に、どの AI が答えたかが表示されます。

	<Screenshot
		src="claude-code-statusline.png"
		alt="Claude Code のフッターに表示された Waired のステータス行。直前の応答を返したローカルモデル名が出ている。"
		caption="どちらの AI が答えたかがフッターでわかります。"
	/>

	```sh
	waired claude route           # 今どちらを使っているか表示
	waired claude route waired    # 自分の AI だけを使う
	waired claude route anthropic # 本来の Anthropic API を使う
	```

	Claude Code の会話中なら `/waired-route` でも切り替えられます。

</Steps>
