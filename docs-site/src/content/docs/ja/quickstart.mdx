このページを最後まで進めると、**自分のパソコンで動く AI** ができあがり、
Claude Code がクラウドではなくその AI に質問するようになります。

## はじめる前に

|  | |
|---|---|
| **かかる時間** | 操作は 10 分ほど。そのあとダウンロードの待ち時間が 5〜30 分あります（放置で OK）。 |
| **必要なもの** | AI を動かすパソコン 1 台（Windows 11 / macOS 13 以降 / Ubuntu 24.04 以降・Debian trixie 以降）、Google アカウント、そのパソコンの管理者権限。 |
| **なくてもいいもの** | グラフィックボード（GPU）。あれば速くなりますが、最近の CPU でも小さめのモデルなら動きます。 |

<Aside type="tip" title="どのパソコンで動かせばいい？">
	いちばん性能の高い 1 台を選んでください（グラフィックボード搭載のデスクトップ、
	または Apple Silicon の Mac）。ノートパソコンは、あとから**どこにいても**
	そのパソコンの AI を使えるようになります。
	パソコンが 1 台しかない場合は、その 1 台で大丈夫です。
</Aside>

## ステップ 1 — Waired をインストールする

インストールはコマンド 1 つです。ターミナルを開き、自分の環境の行を貼り付けて
Enter を押してください。

<Tabs syncKey="os">
	<TabItem label="Windows">
		**管理者として PowerShell を開きます**: スタートボタンを右クリック →
		**「ターミナル（管理者）」** →「はい」。

		```powershell
		iwr -useb https://github.com/waired-ai/waired-agent/releases/latest/download/install.ps1 | iex
		```

		`iwr` が Waired のインストールスクリプトをダウンロードし、`iex` が実行します。
	</TabItem>
	<TabItem label="macOS">
		**ターミナル**を開きます: ⌘ + スペース →「ターミナル」と入力 → Enter。

		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```

		`curl` がスクリプトをダウンロードし、`sh` が実行します。途中で Mac の
		パスワードを聞かれますが、**入力しても画面には何も表示されません**。故障では
		ないので、そのまま入力して Enter を押してください。
	</TabItem>
	<TabItem label="Linux">
		ターミナルを開きます。

		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```

		`curl` がスクリプトをダウンロードし、`sh` が実行します。途中でパスワードを
		聞かれますが、**入力しても画面には何も表示されません**。故障ではありません。
	</TabItem>
</Tabs>

インストーラーは何かを変更する前に、これから行う内容を表示して
`Proceed? [Y/n]` と確認します。Enter を押せば続行します。

<Aside type="tip" title="スクリプトの中身を先に読みたい場合">
	ダウンロードしてから実行できます。各 OS のページを参照してください:
	[Windows](/ja/getting-started/install/windows/) ·
	[macOS](/ja/getting-started/install/macos/) ·
	[Linux](/ja/getting-started/install/linux/)
</Aside>

自分で打つ必要があるコマンドはこれだけです。ここから先は、進め方を選べます。

## ステップ 2 — セットアップを終える

<Tabs syncKey="mode">
	<TabItem label="画面で進める">

		<Steps>

		1. メニューバー（macOS）またはタスクトレイ（Windows / Linux）の
			**Waired アイコン**をクリックし、**Log in…** を選びます。

			<Screenshot
				src="app-not-signed-in.png"
				alt="Waired のメニューが開いた状態。先頭に「○ Not signed in」、その下に「Log in…」の項目が見えている。"
				caption="サインイン前は「Not signed in」と表示されます。"
				width="26rem"
			/>

		2. 開いたブラウザーで **Google アカウントを選んでサインイン**します。

		3. **ブラウザがセットアップを進めます。** AI ソフトの導入、モデルのダウンロード、
			速度の計測が自動で進みます（→ [3 つの工程の中身](#3-つの工程の中身)）。

			<Screenshot
				src="setup-wizard-progress.png"
				alt="ブラウザに表示された Waired のセットアップ画面。Install the AI software（完了）、Download the AI model（19.2 GB 中 7.9 GB で進行中）、Check the speed（待機中）の 3 ステップ。"
				caption="このタブは閉じても大丈夫です — ダウンロードは続きます。"
			/>

		4. コーディングツールの連携を聞かれたら、**Claude Code を有効に**します。

		</Steps>

		<Aside type="caution" title="画面が「コマンドを実行してください」と言ってきたら">
			管理者権限が必要な工程は、ブラウザからは実行できません。その場合は上の
			**「ターミナルで進める」**タブに切り替えて `waired init` から進めてください。
			それまでの進捗は失われません。
		</Aside>

	</TabItem>
	<TabItem label="ターミナルで進める">

		<Steps>

		1. **セットアップを開始します。**

			<Tabs syncKey="os">
				<TabItem label="Windows">
					```powershell
					& "C:\Program Files\Waired\waired.exe" init
					```
					（先ほどの管理者 PowerShell のまま実行します）
				</TabItem>
				<TabItem label="macOS">
					```sh
					sudo waired init
					```
				</TabItem>
				<TabItem label="Linux">
					```sh
					sudo waired init
					```
				</TabItem>
			</Tabs>

		2. **サインインします。** リンクが表示されて止まるので、**Enter** を押すと
			ブラウザーが開きます。Google アカウントを選んでください。

		3. **2 つの質問に答えます。**

			- *Run AI models on this computer?*（このパソコンで AI を動かしますか？）
			  — 上で選んだパソコンなら **Y**。使うだけのノートパソコンなら **N**。
			- *Let your other devices use this computer's AI?*（ほかの端末から使えるように？）
			  — ノートパソコンなどから使いたいなら **Y**。

		4. **待ちます。** AI ソフトの導入、このマシンに合うモデルの選定とダウンロード、
			速度の計測が進みます（→ [3 つの工程の中身](#3-つの工程の中身)）。

			```
			Starting the inference engine…
			Preparing to download qwen3-coder-30b-a3b-instruct…
			Downloading qwen3-coder-30b-a3b-instruct: 41%  7.9 GB / 19.2 GB (44 MB/s)
			```

		5. Claude Code を Waired 経由にするか聞かれたら **Y** を選びます。

		</Steps>

		**こう表示されれば完了です**:

		```
		╭─ 🎉 Waired is ready — everything completed successfully! ─────────╮
		│  Account   you@example.com                                       │
		│  Device    my-desktop  d-4f2a…                                   │
		│  Network   personal  overlay 100.64.0.3                          │
		│  Model     qwen3-coder-30b-a3b-instruct  (58 tok/s)              │
		│  Gateway   http://127.0.0.1:9479                                 │
		│  Claude    http://127.0.0.1:9473 (managed settings)              │
		│                                                                  │
		│  Point your coding agent at Waired and start building.           │
		╰──────────────────────────────────────────────────────────────────╯
		```

	</TabItem>
</Tabs>

<Aside type="note" title="どちらか一方に決める必要はありません">
	同じセットアップを 2 か所から見ているだけです。ターミナルから始めてもブラウザの
	画面は開きますし（*「Setup is continuing in your browser…」*で **Enter** を押せば
	ターミナルに戻れます）、片方での変更はもう片方にも反映されます。画面のないサーバーでは
	ターミナル一択です。
</Aside>

### 3 つの工程の中身

| 工程 | 何をしているか | 目安 |
|---|---|---|
| **Install the AI software** | モデルを読み込んで動かすプログラムの導入。管理者権限が必要です。 | 1〜2 分 |
| **Download the AI model** | AI 本体。このパソコンで実際に動かせる最良のモデルを Waired が選ぶので、自分で選ぶ必要はありません。 | 5〜30 分 |
| **Check the speed** | 速度の計測。追いつかない場合は、より軽いモデルが提案されます。 | 最大 5 分 |

## ステップ 3 — 試しに話しかける

<Tabs syncKey="mode">
	<TabItem label="画面で進める">
		もう一度 **Waired アイコン**をクリックします。アカウント名が表示され、
		**Inference** に動作中のモデル名が出ていれば成功です。

		<Screenshot
			src="app-ready.png"
			alt="Waired のメニュー。サインイン済みのアカウント、接続状態、Inference サブメニューに動作中のモデル名が表示されている。"
			caption="サインイン済み・接続済み・モデル読み込み済み。これが正常な状態です。"
			width="26rem"
		/>
	</TabItem>
	<TabItem label="ターミナルで進める">
		```sh
		waired infer "say hi"
		```

		数秒であいさつが返ってきます。この返事は**あなたのパソコンの中だけ**で
		作られたもので、クラウドの AI サービスには何も送られていません。
	</TabItem>
</Tabs>

<Aside type="note" title="何も返ってこないとき">
	`waired doctor` を実行してください。設定を上から順に点検して ✓ / ⚠ / ✗ で表示し、
	**f** キーを押すと直せるものは自動で直します。初回のつまずきはこの 1 コマンドで
	ほぼ解決します。
</Aside>

## ステップ 4 — Claude Code から使う

いつもどおり Claude Code を開いて質問してみてください。
画面下部に、どの AI が答えたかが表示されます。

<Screenshot
	src="claude-code-statusline.png"
	alt="Claude Code のフッターに表示された Waired のステータス行。直前の応答を返したローカルモデル名が出ている。"
/>

会話中に `/waired-route` と入力すれば自分の AI とクラウドを切り替えられます。
ターミナルからでも同じことができます:

```sh
waired claude route           # 今どちらを使っているか表示
waired claude route waired    # 自分の AI だけを使う
waired claude route anthropic # 本来の Anthropic API を使う
```
