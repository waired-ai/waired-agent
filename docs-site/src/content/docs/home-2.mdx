## Install, in one command

Every platform installs the same way: paste one line into a terminal. It shows
you what it will do and asks before changing anything.

<Tabs syncKey="os">
	<TabItem label="Windows">
		```powershell
		iwr -useb https://github.com/waired-ai/waired-agent/releases/latest/download/install.ps1 | iex
		```

		From **Terminal (Administrator)** — right-click the Start button.
	</TabItem>
	<TabItem label="macOS">
		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```

		From **Terminal** — press ⌘ + Space and type `Terminal`. macOS 13+.
	</TabItem>
	<TabItem label="Linux">
		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```

		Ubuntu 24.04+ / Debian trixie+.
	</TabItem>
</Tabs>
