Waired connects your laptops, GPU servers, VMs and home boxes so that
**local AI models can be shared across your own devices** without exposing them
to the public internet. Type in Claude Code on any laptop and the answer comes
from your own machine — no per-message bill, no subscription, and your prompts
never leave your devices.

## Install in one command

<Tabs syncKey="os">
	<TabItem label="Windows">
		From **Terminal (Administrator)** — right-click the Start button:

		```powershell
		iwr -useb https://github.com/waired-ai/waired-agent/releases/latest/download/install.ps1 | iex
		```
	</TabItem>
	<TabItem label="macOS">
		From **Terminal** — press ⌘ + Space and type `Terminal`:

		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```
	</TabItem>
	<TabItem label="Linux">
		From a terminal, on Ubuntu 24.04+ or Debian trixie+:

		```sh
		curl -fsSL https://github.com/waired-ai/waired-agent/releases/latest/download/install.sh | sh
		```
	</TabItem>
</Tabs>

It shows what it will do and asks before changing anything. Then finish setup
**on screen or in the terminal** — the [Quickstart](/quickstart/) covers both.
