Waired installs with **one command**, the same way on all three platforms: you
paste a line into a terminal, it shows you what it will do, and you confirm.
There is no installer to download and double-click.

## Will it run on my computer?

| Role | What works |
|---|---|
| **Runs the AI** | Ubuntu 24.04+, Debian trixie+, Windows 11 / Server 2025, macOS 13+ (Apple Silicon recommended) |
| **Uses the AI** | Any of the above — no graphics card needed |

A graphics card makes answers much faster, but a recent processor runs a small
model fine. The computer you *type* on needs nothing special.

How much memory a given model needs is in the
[model catalog](/reference/model-catalog/) — though you do not have to read it,
because Waired picks a model that fits when you set up.

## What the installer puts on your computer

- **The Waired app** — a background service that keeps this computer connected,
  and a `waired` command you can type. The service starts with the computer,
  before anyone logs in.
- **The Waired app** — an icon in the menu bar on macOS, next to the clock on
  Windows and Linux, for
  everyday controls.
- **Nothing else, yet.** The AI software itself (a multi-gigabyte download) is
  installed later, during setup, and only if you say this computer should run
  models.

Every installer prints a summary of what it is about to do and waits for you to
confirm before touching anything.

<Aside type="note" title="Already run Ollama?">
	Waired can leave your existing installation alone. See
	[Advanced install options](/reference/install-options/).
</Aside>
